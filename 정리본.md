

### cp 7 - 완전탐색 2 
    * 물체 단위로 완전 탐색

    ``` n = 5
segments = [(1, 3), (2, 4), (5, 8), (6, 9), (7, 10)]

ans = 100
for i in range(n):
    count = [0] * 11 # initialize
    for j in range(n):
        if j == i:
            continue

        x1, x2 = segments[j]
        for k in range(x1, x2 + 1):
            count[k] += 1

    max_cnt = max(count)
    ans = min(ans, max_cnt)

print(ans)
``` 
